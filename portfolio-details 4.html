Machine Learning Mini Projects & Experiments

This portfolio entry represents a curated collection of various machine learning mini-projects and data science experiments I have conducted. These projects serve as a practical exploration of different algorithms, data preprocessing techniques, and model evaluation metrics. They are primarily built and documented using Google Colab notebooks.

The goal of this collection is to demonstrate versatility in handling different types of data (structured, text, image) and effectively applying the appropriate ML techniques to solve specific problems.

Project Collection Highlights

  • Predictive Analytics: Regression models for housing price prediction and sales forecasting.
  • Classification Tasks: Customer churn prediction and iris flower classification using Support Vector Machines (SVM) and Random Forests.
  • Clustering: K-Means clustering for customer segmentation analysis.
  • Natural Language Processing (NLP): Sentiment analysis on review datasets using basic nltk and scikit-learn pipelines.

Tools & Libraries

The projects extensively use the Python data science ecosystem:

  • Pandas & NumPy: For data manipulation and numerical analysis.
  • Matplotlib & Seaborn: For data visualization and exploratory data analysis (EDA).
  • Scikit-Learn: For model building, training, and evaluation.
  • Google Colab: Cloud-based notebook environment for development.

Each mini-project was an opportunity to deep dive into a specific algorithm. From understanding the math behind Gradient Descent to fine-tuning hyperparameters in a Neural Network, this collection documents my continuous journey in mastering AI.
