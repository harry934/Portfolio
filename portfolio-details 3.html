Campus Events & Student Management System

The Campus Events System is a comprehensive web application developed using PHP and MySQL designed to streamline the management of school events, club registrations, and student participation. This project addresses the need for a centralized platform where students can stay informed about campus activities and administrators can easily manage logistics.

This system replaces traditional manual notice boards and paper-based registrations with a digital, interactive solution. It enhances communication between the student body and the administration, fostering a more engaged campus community.

Key Features

  • Event Management: Administrators can create, update, and delete campus events with detailed schedules.
  • Club Registration: Students can browse available clubs and register online entirely paperless.
  • Student Dashboard: A personalized area for students to track their registered events and club memberships.
  • Admin Panel: Robust backend for managing users, approving registrations, and analytics.
  • Calendar Integration: Visual calendar view of upcoming events for easy planning.

Technical Architecture

The application follows the MVC (Model-View-Controller) architectural pattern to ensure code maintainability and separation of concerns.

  • Frontend: HTML5, CSS3, Bootstrap for responsive layout.
  • Backend: PHP (Vanilla) for server-side logic and session management.
  • Database: MySQL for storing user data, events, and registrations efficiently.
  • Security: Implemented prepared statements to prevent SQL Server injection and secure password hashing.

This project was a pivotal learning experience in full-stack web development. It taught me the importance of database normalization, secure coding practices, and user-centric design in building functional web applications.
